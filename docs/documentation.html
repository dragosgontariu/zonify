Zonify Documentation

Complete guide to using Zonify for advanced zonal statistics analysis in QGIS.

Getting Started

Zonify is a professional QGIS plugin for performing advanced zonal statistics analysis. It allows you to efficiently extract and analyze raster data within polygon zones.

‚ú® What makes Zonify special?

  • Process unlimited rasters in one batch
  • 15+ statistical measures including coverage analysis
  • Multi-threaded processing with smart resource management
  • Professional HTML dashboards and PDF reports
  • Advanced tools: Score Creator, Classifier, Map Generator

Installation

Requirements

  • QGIS: 3.28 or higher
  • Python: 3.9 or higher
  • RAM: 4GB minimum, 16GB recommended
  • CPU: 2 cores minimum, 4+ recommended

Step 1: Download Plugin

Download the latest release from GitHub Releases.

Step 2: Install to QGIS

Extract the ZIP file to your QGIS plugins directory:

  • Windows: C:\Users\<username>\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\
  • Linux: ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/
  • macOS: ~/Library/Application Support/QGIS/QGIS3/profiles/default/python/plugins/

Step 3: Install Python Dependencies

Windows (OSGeo4W Shell as Administrator):

pip install numpy pandas scipy matplotlib plotly reportlab jinja2 psutil pyarrow

Linux/Mac:

pip3 install --user numpy pandas scipy matplotlib plotly reportlab jinja2 psutil pyarrow

Step 4: Enable Plugin

  1. Restart QGIS
  2. Go to Plugins ‚Üí Manage and Install Plugins
  3. Find "Zonify" and check the box
  4. Look for the Zonify icon in the toolbar

Quick Start

Get started with Zonify in 5 minutes!

Basic Workflow

  1. Open Zonify: Click the Zonify icon or go to Plugins ‚Üí Zonify
  2. Select Polygon Layer: Choose your zones/regions layer
  3. Add Rasters: Click "Add Raster(s)" and select your files
  4. Choose Statistics: Select which statistics to calculate (Mean, Sum, etc.)
  5. Set Output: Choose where to save results
  6. Run Processing: Click "Run Processing" button

üí° Pro Tip

Start with a small dataset (10-20 polygons, 1 raster) to test the workflow before processing large datasets.

Core Features

Statistical Measures

Zonify calculates 15+ statistical measures:

  • Position: Mean, Median, Mode, Min, Max
  • Dispersion: Sum, Standard Deviation, Variance, CV, Range
  • Distribution: Percentiles (25th, 50th, 75th, 90th, 95th)
  • Coverage: Pixel count and coverage percentage (pixel-perfect geometric analysis)

Coverage Analysis

Coverage shows what percentage of each polygon contains valid raster data. This is crucial for:

  • Data quality assessment
  • Identifying incomplete coverage
  • Understanding NoData areas
  • Making informed decisions about statistical validity

Batch Processing

Process multiple rasters simultaneously with:

  • Multi-threaded processing (use all CPU cores)
  • Smart RAM management
  • Resume/checkpoint system
  • Background processing (QGIS stays responsive)

Advanced Tools

Score Creator

Combine multiple indicators into a single composite score with weighted importance levels.

Use cases: Risk assessment, suitability analysis, priority ranking

Area Classifier

Classify continuous values into categories using:

  • Equal Intervals
  • Quantiles
  • Natural Breaks (Jenks)
  • Custom breakpoints

Quick Map Generator

Generate professional maps with:

  • Automatic legend
  • Scale bar
  • North arrow
  • Custom title and subtitle
  • Export to PNG/PDF

Time Series Analysis

Analyze temporal changes with:

  • Trend detection
  • Change analysis
  • Temporal statistics

Export Formats

  • GeoPackage (.gpkg): Primary output with all attributes
  • CSV: Tabular data for Excel/spreadsheets
  • JSON: Structured data for APIs
  • HTML: Interactive dashboards with charts
  • PDF: Professional printable reports

Troubleshooting

Common Issues

‚ùå "Module not found: numpy"

Solution: Python dependencies not installed. Run pip install command from installation section.

‚ùå Processing crashes or freezes

Solutions:

  • Reduce RAM limit in settings
  • Process fewer rasters at once
  • Check for corrupted raster files
  • Validate polygon geometries

‚ùå Results seem incorrect

Checks:

  • Verify CRS matches between polygons and rasters
  • Check raster NoData values
  • Review coverage statistics
  • Verify units are correct
